24 自定义组件

	2 js:
		a Component({}) ：创建一个component对象
		b properties : 设置component对象上的属性，这里的属性值设置不能没有定义，不然报错，定义的语法和data还有点不一样
			1 变量：绑定数据 pro-a，pro-b,用户调用处，动态绑定数据
		c data{}:私有数据，用户渲染模板，和调用处的data数据域，互相独立
		d methods{}: 组件自己业务逻辑
		e options{}:组件的配置，组件自身有的一些属性，可以通过在这里配置起作用：
			1 multipleSlots: true // 在组件定义时的选项中启用多slot支持
			2 styleIsolation: 'isolated' //表示启用样式隔离
		f externalClasses: ['class'], // 可以将 class 设为 externalClasses,引用外部样式
	3 wxml:<view><slot></slot></view>
	4 引用：wxml:<component-tag-name></component-tag-name>,
			json:
				{
					"usingComponents":

					}
				}
